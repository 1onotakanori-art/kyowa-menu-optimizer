# 実行方法クイックガイド

メニュースクレイピングの実行方法を簡潔に説明します。

## 準備

### SSH キーの設定（初回のみ）

```bash
# GitHub との接続テスト
ssh -T git@github.com
```

成功すれば OK です。

---

## メニュースクレイプ & アップロード

**目的:** メニューをスクレイピングして menus/ に保存 + Supabase へアップロード

```bash
# 5日分（デフォルト）
npm run scrape

# 10日分
npm run scrape:10

# 20日分
npm run scrape:20
```

**何が起こる？**
- Kyowa のサイトからメニュー取得
- `menus/` に JSON として保存
- Supabase へ自動アップロード

**実行タイミング:**
- メニューが更新されたとき
- 推奨: 週に1回（月曜日など）

---

## エラーが出た場合 "日付が見つかりません"

サイトにメニューが登録されていない可能性があります。翌日再実行してください。
